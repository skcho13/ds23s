이번 장에서는 시각화를 하기 전, 중요한 데이터 패턴을 보기 위해서 새로운 변수를 만들거나 요약한 통계치를 만들 필요가 있는데 이를 다루는 핵심적인 함수들에 대해 익힙니다.
좀 더 자세한 데이터 가공에 대해서는 추후에 다룰 예정입니다.

대략 다음과 같은 transform들을 조합하여 분석에 필요한 상태로 바꿉니다.

  • 변수들의 선택: subsetging
  • 조건에 맞는 부분(관측치, rows)만 필터링: query()
  • 조건에 맞도록 columns을 재정렬: sort_values()
  • 변수들과 함수들을 이용하여 새로운 변수를 생성: assign()
  • 카테고리별로 나뉘어진 데이터에 대한 통계치를 생성: groupby(), agg()

Apply: General split-apply-combine McKineey’s Chapter 10.3 참고

The most general-purpose GroupBy method is apply, which is the subject of this section.
apply splits the object being manipulated into pieces, invokes the passed function on each piece, and then attempts to concatenate the pieces.
